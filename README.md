# rime-bin-parser
Use Java to parse the various bin files generated by Rime

### Windows 加载动态链接库问题
> [!CAUTION]
> 如果出现"java.lang.UnsatisfiedLinkError: xxx.dll: Can‘t find dependent libraries"，大概率是因为当前所使用的动态链接库 dll 丢失了依赖库，比如 [actions](https://github.com/12302-bak/marisa-java/blob/master/.github/workflows/verify-native.yml) 中运行在 **windows-2025** 平台上的环境就缺少`msvcp100.dll`依赖，需要单独安装才可正常运行。
>
> 需要排查当前工作环境中缺少的依赖，可以使用工具 [Dependencies](https://github.com/lucasg/Dependencies)(进阶版 Dependency Walker) 进行排查。
